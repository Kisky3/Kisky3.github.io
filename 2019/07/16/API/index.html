jQuery常用API

.append()

添加元素，通过$符号，生成一个dom元素并将它添加到页面，也可以添加jQuery对象，字符串等

.prepend()

在元素的前头添加字符串或元素。

.before()

在对象前面（不是内部而是外面，和对象同级）插入内容，参数和append相似。

.after()

在对象后面（不是内部而是外面，和对象同级）插入内容，参数和append相似。

.remove()

删除所选对象的元素或者子元素。

.empty()

删除指定元素内的所有子元素。该元素保留

.html()

实用html()能够任意的获取HTML元素，并进行添加或替换处理。
