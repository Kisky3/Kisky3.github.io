常见npm错误

npm install error

在npm install时候发生error的话可以检查以下：

  1. 是否存在package.json,因为npm安装的时候是根据package.json获取相应包的。没有源文件的话就无法获取资源。

  1. 在有package.json的情况下有可能是重复多次安装npm，导致冲突。
    可以尝试以下步骤后再次安装。

  2. 当出现以下error的时候说明端口被占用,可以修改端口或者kill其他占用此端口的程序。
