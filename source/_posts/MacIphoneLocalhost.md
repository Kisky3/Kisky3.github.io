用iphone连接Mac的Localhost服务器
<!--more-->

1.iphone用USB连接Mac

2.**iphone和Mac连接同一个wifi**

3.Mac ： System Setting => Network 查看该网络下的ip地址比如192.168.100.xxx

4.iphone地址栏输入ip地址 + port号即可以预览localhost

```
 192.168.100.xxx:8080
 ```
